## 🚀 Getting Started
  - clone the repo <br>
  `git clone <repo> my-tweeter`
  - Navigate to the repo<br>
  `cd my-tweeter`
  - Install all the depndencies <br>
  `npm install`
  - Run the server: <br>
  `npm run local`  <br>
  - Visit the app in your browser: `http://127.0.0.1:8080/` 
  - Start Tweeting 🐥🎉🎇
